Research Adventures

Since January 2024, I've been a Machine Learning Research Intern at the Zhou Lab at UT Southwestern Medical Center, under the guidance of Dr. Jian Zhou and Dr. Kseniia Dudnyk. There, I created an interpretable deep learning model to predict transcription start site locations in Drosophila, uncovering how promoter motifs influence genetic expression. Currently, I'm training models to forecast enhancer-promoter interactions and DNase I Hypersensitive Sites, leveraging high-performance computing clusters with A100 and H100 GPUs for distributed training.

This work has sharpened my skills in PyTorch, GPU computing, genomic sequence modeling, and data visualization—turning raw biological data into meaningful insights. It's exhilarating to contribute to fields that could reshape our understanding of genetics.

Key Projects That Shaped Me

In 2022, as part of CMU's Deep Learning Systems course, I built the Needle Deep Learning Library from the ground up— a PyTorch-like framework using Python, C++, and CUDA. I implemented automatic differentiation, optimizers like SGD and Adam, normalization layers, and GPU acceleration, culminating in adding Graph Neural Network support for classification tasks. It was a hands-on lesson in efficient tensor operations and tiled matrix multiplication.

For the Texas JSHS in 2023 (where I placed in the Top 8 for Life Sciences), I developed a system for Single Cell Gene Expression Prediction, tackling high-dimensional datasets in computational biology. I combined ensemble methods with ML approaches for perturbation prediction and built an automated hyperparameter tuning pipeline using Optuna—blending PyTorch, HuggingFace, and statistical modeling.

Earlier, in my MIT 6.86x final project, I created a Reinforcement Learning Game Agent, implementing Q-learning variants and Deep Q-Networks to build an autonomous agent in a text-based environment. These projects aren't just code; they're stories of perseverance and innovation.
