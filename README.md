# iishaan-i.github.io

Personal website for Iishaan Inabathini.

## Structure

- `/` (index.html) - Simple home page with name and class
- `/notes/` - ML paper notes
- Navigation available on both pages

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit:
- Home: `http://localhost:4000`
- Notes: `http://localhost:4000/notes/`

## Deployment

Push to GitHub and it will auto-deploy to `https://iishaan-i.github.io`

Live URLs:
- Home: `https://iishaan-i.github.io/`
- Notes: `https://iishaan-i.github.io/notes/`
